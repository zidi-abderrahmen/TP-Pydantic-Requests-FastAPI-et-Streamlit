# Advanced Python - Practical Works

This repository contains the first 4 practical works (TPs) for the Advanced Python course at Teknolabs.

## 📁 Repository Structure
```
TP-Pydantic-Requests-FastAPI-et-Streamlit/
├── 1-pydantic/          # TP1: Data validation with Pydantic
├── 2-requests/          # TP2: HTTP requests handling
├── 3-fastapi/           # TP3: API development with FastAPI
├── 4-streamlit/         # TP4: Web apps with Streamlit
└── README.md
```


## 🚀 TPs Overview

| TP | Topic | Description |
|----|-------|-------------|
| 1 | [Pydantic](https://docs.teknolabs.net/courses/advanced-python/1-pydantic/) | Data validation and settings management using Python type hints |
| 2 | [Requests](https://docs.teknolabs.net/courses/advanced-python/2-requests/) | Making HTTP requests in Python |
| 3 | [FastAPI](https://docs.teknolabs.net/courses/advanced-python/3-fastapi/) | Building modern, fast web APIs with FastAPI |
| 4 | [Streamlit](https://docs.teknolabs.net/courses/advanced-python/4-streamlit/) | Creating data apps in pure Python |

## 🛠️ Requirements

- Python 3.8+
- See individual TP folders for specific dependencies
